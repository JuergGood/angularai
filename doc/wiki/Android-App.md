# Android Application

The AngularAI project includes a native Android companion app built with modern development tools.

## Key Technologies

- **Jetpack Compose**: For a modern, declarative UI.
- **Kotlin**: The primary programming language.
- **Retrofit**: For API communication with the Spring Boot backend.
- **Material 3**: For a consistent design language.

## Features

- **Task Management**: View and update tasks on the go.
- **User Authentication**: Secure login to the same backend used by the web app.
- **Real-time Updates**: Stay synchronized with your task list.

## Development Setup

The Android project is located in the `android/` directory.

1. Open the `android/` folder in **Android Studio**.
2. Sync the project with Gradle files.
3. Run the app on an emulator or a physical device.

For more detailed build instructions, see the `doc/ai/android/android-build-instructions.md` file in the main repository.
